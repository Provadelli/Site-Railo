# NEXURA

Uma landing page moderna para uma plataforma **HR-Tech Marketplace**, conectando empresas aos melhores profissionais de Recursos Humanos de forma simples, rápida e segura.

## Sobre

A NEXURA foi desenvolvida com foco em uma experiência moderna, responsiva e intuitiva, apresentando os principais recursos da plataforma através de uma interface limpa e elegante.

## Funcionalidades

- Landing page totalmente responsiva
- Interface moderna
- Modo Claro/Escuro
- Marketplace de profissionais
- Seção de planos
- Depoimentos
- FAQ interativo
- Menu responsivo
- Animações suaves
- Componentes reutilizáveis

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Estrutura


## Como executar

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/nexura.git
```

Entre na pasta:

```bash
cd nexura
```

Abra o arquivo `index.html` no navegador.

## Objetivo

Demonstrar o conceito de uma plataforma digital voltada para o mercado de Recursos Humanos, utilizando uma interface moderna e de alta qualidade.
